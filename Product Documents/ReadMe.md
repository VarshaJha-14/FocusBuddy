# 📂 Product Documents

This folder contains the product management artifacts created during the discovery, strategy, design, and validation of **FocusBuddy**.

Rather than showcasing only the final solution, these documents capture the product thinking behind it, including user research, problem definition, prioritization, UX design, AI strategy, and roadmap planning.


## Objective

These artifacts demonstrate an end-to-end product management process, from identifying a user problem through designing, validating, and planning an AI-powered solution.

> **Note:** FocusBuddy is a portfolio project created to demonstrate Product Management, AI Product Strategy, UX thinking, and MVP execution.
